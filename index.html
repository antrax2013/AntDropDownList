AntDropDownList library


v0.1 - 05/31/2014

AntDropDownList, is a custom DropDownList jQuery plugin.

    2 avaible types of AntDropDownList:
  • AntDropDownList: basic trunkey drop down list.
  • AntPairedDropDownList: master and slave drop down lists. Slave drop down list is filtred on change event of master drop down list fired. [avaible soon]

AntDropDownList

It's basic trunkey drop down list. Elements are loaded by ajax.

Methods Parameters

Single method to fill drop down list by ajax 		$cible [jQuery handdle] A jQuery handdle of selected tag of DOM. In this version, the selected tag must be a select html tag.

withEmptyEntry [optionnal boolean] Boolean to define if empty entry is required. False by default
selectedValue [optionnal string] The selected value. Ignore by default
